## Description
The main goal of the `get_next_line` project is creating a function that returns one line at a time from a file descriptor. This challenge allows you to dive deeply into key Unix concepts such as **file descriptors**, the `open()`, `read()` and `close()` system calls, and **static variables** to persist state between function calls.

The implementation handles any line length and works efficiently with large files by reading in fixed-size chunks defined by `BUFFER_SIZE` at compile time.
